考研政治 - 智能复习卡片 (基础版)

这是一个极简的、用于考研政治的智能记忆卡片 Web 应用。它专注于提供一个连续不断的复习流程，帮助用户通过间隔重复算法 (Spaced Repetition) 高效记忆。

本项目是一个纯粹的单 HTML 文件，不依赖任何服务器，下载后在浏览器中打开即可立即使用。

✨ 主要功能

间隔重复算法：内置艾宾浩斯遗忘曲线逻辑。根据你对卡片“不记得”、“困难”、“记得”的评级，自动计算下一次最佳复习时间 (T+1天, 3天, 7天...)。

智能抽卡逻辑：

优先复习：优先抽取所有“到期”的卡片。

学习新卡：当没有到期卡片时，系统会自动抽取 1 张“新卡片”供你学习。

无限复习：当所有卡片都已学习且均未到期时，系统会从全部卡片中随机抽取进行“无限复习”，确保知识的持续巩固。

实时进度统计：在页面顶部实时显示“未学习”、“易遗忘”和“已掌握”的卡片总数。

数据持久化：所有学习进度自动保存在浏览器的 localStorage 中。

数据重置：提供“重置学习进度”按钮，方便重新开始。

响应式设计：界面适配桌面和移动设备。

🚀 如何使用

下载本仓库中的 test2.html 文件。

在你的电脑或手机浏览器中直接打开这个 .html 文件。

点击“开始学习”按钮，进入连续复习流程。
